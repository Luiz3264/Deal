# Deal

A simple POS made from scratch with React.js, Tailwind CSS, Vite and Typescript.

## Instructions

Use `vite` or `npm run dev` in the root directory after installing the dependencies to start the web server.

Edit `src/data.csv` to modify the products list. First the bar code, then the name and finally the price.

The default config password is: `1234`
